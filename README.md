# MaBeeeForSchool_Control_Eye

重度肢体不自由児・者が視線入力デバイス（Tobii Eye Tracker 5など）やアクセシビリティスイッチ入力でマウスクリックできるインターフェイスを使用して、MaBeee for Schoolを内蔵した乾電池式オモチャなどを操作するためのWindowsアプリです。タイマー実行やカメラ表示機能を搭載しています。

## 概要

このソフトウェアは、視線入力やスイッチを利用して乾電池駆動の玩具等を制御することを目的としています。「MaBeee for School」との連携により、Bluetooth経由で出力をコントロールします。

## 主な機能

- 視線入力デバイス対応（マウスクリック代替）
- アクセシビリティスイッチによる操作
- タイマー実行機能
- PCカメラ表示（操作対象の確認用）

## 使い方

1. `MaBeeeForSchool_Control_Eye.exe` を実行します。
2. アプリ上で「MaBeee」を検索し、接続します。
3. 接続後、画面上のボタンまたは設定したインターフェースで玩具を操作できます。

詳細な使い方は、[かんたんガイドMaBeeeForSchool_Control_Eyeアプリ.pdf] をご覧ください。
